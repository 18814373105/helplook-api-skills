# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-03-18

### Added

- `SKILL.md` - Agent Skill 主文件
- `scripts/` - Python 可执行脚本
  - `mcp_client.py` - API 客户端核心
  - `article_list.py` - 获取文章/目录列表
  - `article_search.py` - 关键词搜索
  - `article_get.py` - 获取文章详情
  - `article_create.py` - 新建文章/目录
  - `article_update.py` - 修改文章/目录
  - `article_update_content.py` - 仅更新正文
  - `article_publish.py` - 发布草稿
  - `article_delete.py` - 删除文章
  - `article_version_list.py` - 版本列表
  - `article_hot_list.py` - 热门文章列表
- `references/` - 工具参数参考文档
- `examples/` - 调用示例
- `api-key.json.example` - API 配置模板
- `pyproject.toml` - Python 项目配置
- `requirements.txt` - 依赖声明
- `CONTRIBUTING.md` - 贡献指南
- `LICENSE` - MIT 许可证
